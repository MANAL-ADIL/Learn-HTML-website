HTML tag kiya h?

ANGULAR BRACKETS(< >):

ANGULAR BRACKETS ko hum HTML me tags likhne ke liye use karte hain. Ye brackets hamesha opening tag (< >) aur closing tag (< / >) ke liye istemal hote hain.

HTML kaise kaam krta h:

HTML kaam krta h tags ke zariye. Tags ko hum anglular brackets (< >) me likhte h. Har tag ka ek opening tag hota h or ek closing tag hota h. Opening tag me hum element ka naam likhte h or closing tag me bhi wahi naam likhte h lekin isse pehle slash (/) lagate h. Jaise ke <h1>opening tag h or </h1> closing tag h. Is trha se hum HTML me elements create krte h.
