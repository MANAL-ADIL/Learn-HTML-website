VISUAL STUDIO CODE

How to install visual studio code

1. Go to the official website of Visual Studio Code.

2. Click on the "Download" button for your operating system (Windows, macOS, or Linux).

3. Once the installer is downloaded, run it and follow the installation instructions.

4. After installation, launch Visual Studio Code.

Install Extension

1. Open Visual Studio Code.

2. Go to the Extensions view by clicking on the Extensions icon which is on 5th position in the activity bar.

3. Search for the LIVE SERVER in the Marketplace search bar.

4. Click on the "Install" button to install the LIVE SERVER.
