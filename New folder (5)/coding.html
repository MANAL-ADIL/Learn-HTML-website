FRONTEND & BACKEND DEVELOPMENT:

Frontend Development

Frontend development se murad un tamam cheezon se hai jo user ko dikhai deti hain jab wo kisi website ya application ka istemal karte hain. IsmeHTML, CSS, aurJavaScript ka istemal hota hai taake user interface ko design kiya ja sake. Frontend developer ka kaam hota hai ke wo website ko aisa banaye ke wo user-friendly ho aur uska design attractive ho.Usermeans wo insan jo website ko use kr raha h.HTML frontend dev ke liye istamal hoti h.Frontend dev ke liye hme mostly in 3 languages ki zarurat prti h.


HTML:Structure or content ke liye jo user ke samne ata h.
CSS:Design aur layout ke liye jo user ke samne ata h.
JavaScript:Interactivity aur dynamic behavior ke liye jo user ke samne ata h.

Backend Development

Backend development se murad un tamam cheezon se hai jo user ko nahi dikhai deti, lekin website ya application ke liye bohot zaroori hoti hain. Isme server, database, aur application logic shamil hote hain. Backend developer ka kaam hota hai ke wo server-side scripting, database management, aur application logic ko develop kare taake frontend aur backend ke darmiyan communication ho sake.Backend dev ke liye hm python etc languages use krte h pr filhal ke liye hmara is se kam nhi h.
