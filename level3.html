FORM TAGS

Form tags are used to create interactive forms in HTML. They allow users to input data that can be sent to a server for processing. The main form tag is <form>.Isme action attribute hai jo ke form ko submit karne ke liye istemal hota hai.lkn filhal html me koi server nahi hai isliye hum isse sirf front end par dekh sakte hain.

EXAMPLE:

<form action="submit.php" method="post"> </form>

HOW TO MAKE FORMS

Forms banane ke liye humein kuch basic tags ki zaroorat hoti hai:

Input tag:

Input tag forms bnane ke liye istsmal hota h.Ye ek self closing tag h mtlb isme closing tag nhi hota or jesa ke ap jante h ke attributes opening tag ke andr ate h.

ATTRIBUTES of Input Tag:

type attribute text ho to user se text me input liya jata h agr apka type email h to ap isme email likh skte h phone h to number likh skte h date h to date kuch b likh skte h.

name attribute se input ki value ko identify kiya jata h ke input h kis chez ka jese agr me name me user name likh dun to server ko pta chl jaye ga ke ye user name h.

placeholder attribute se input field me default text dikhaya jata h default text means wo text jo tb dikhta h jb user ne kuch na likha ho or jese hi user kuch likhe to wo chla jaye.

required attribute se input field ko mandatory banaya jata h yani user ko form submit karne se pehle is field me kuch na kuch likhna hi padega.

valueattribute isi liye kiya jata h ta ke server ko pta chl jaye ke ye form kis chez ke liye h.

id attribute se input field ko uniquely identify kiya jata h ta ke agar multiple input fields hain to unhe alag alag identify kiya ja sake. LIKE THIS:

Drop down:

Drop down list banane ke liye humein <select> tag ka istemal karna hota hai. Iske andar hum <option> tags ka istemal karte hain jismein har option ko define kiya jata hai or option tag ke andr hm apna data likhte h.


LIKE THIS:
<select name="subject" required>
<option value="biology">Biology</option>
<option value="physics">Physics</option>
<option value="chemistry">Chemistry</option>
</select>

Textarea:

Textarea banane ke liye humein <textarea> tag ka istemal karna hota hai. Iske andar hum apna data likhte hain or ye ek multi-line text input field provide karta hai jisme hm apna feedback wagera de sakte hain.



LIKE THIS:
<textarea name="feedback" placeholder="Write your feedback here..." rows="5" cols="30" required> </textarea> In this rows attribute represent rows and cols attribute represent columns
